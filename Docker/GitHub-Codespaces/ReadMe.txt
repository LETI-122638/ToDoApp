Devido às limitações do GitHub Codespaces, não é possível aceder ao Codespace criado por outros utilizadores, mesmo que estes pertençam ao mesmo repositório. Cada Codespace é privado e apenas o utilizador que o criou tem permissão para o visualizar e abrir.

Assim, para cumprir os requisitos da ficha e em conformidade com as instruções do docente, um dos elementos do grupo criou o Codespace no repositório da ToDoApp e executou os passos necessários. Esse elemento capturou as imagens pedidas e colocou-as na pasta indicada no GitHub.
